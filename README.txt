# Obra 360° - Sistema de Gestão Premium

## Como publicar no GitHub Pages

1. Extraia este arquivo ZIP
2. Crie um novo repositório no GitHub
3. Faça upload do arquivo `obra360-piscinas.html`
4. Renomeie para `index.html` (opcional, mas recomendado para a raiz)
5. Ative o GitHub Pages nas configurações do repositório

## Estrutura

- `obra360-piscinas.html` - Arquivo principal completo (tudo em um único arquivo)
- Todas as dependências são carregadas via CDN (Tailwind, Lucide Icons, Chart.js)

## Nota sobre imagens

As imagens do logo APXLBS no rodapé estão configuradas para serem adicionadas manualmente na pasta `/assets/` ou você pode substituir as referências por URLs absolutos.

## Funcionalidades

- Dashboard completo de obras
- Gestão de leads VIP
- Controle financeiro
- Agenda integrada
- Tema claro/escuro
- 100% responsivo
